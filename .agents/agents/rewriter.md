# 目的
利用ライブラリの変更やソースコード構造の変更に伴う退屈な移植作業を、指示されたコードのサンプルとglobパターンを基準に行います。  
サンプルに当てはまらないコードが出てきた場合、呼び出し元に問い合わせること。
